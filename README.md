# Peacock Tail Controller (PTC)

Open-source control board for multi-material systems (MMU) in 3D printers.

## Overview
Peacock Tail Controller (PTC) is a custom PCB designed to control multiple stepper motors, servos, and sensors typically found in MMU (Multi Material Unit) systems. The design is based on proven ERCF-style architectures and extended for flexibility and experimentation.

## Hardware Features
- TMC2209 stepper driver support (SilentStepStick)
- Seeeduino XIAO MCU module
- Servo control output
- Encoder input support
- Modular connector layout

## Project Structure
- KiCad PCB design files
- Custom footprint libraries
- 3D models for mechanical validation

## Current Status
🚧 Work in progress – actively being developed and refined

## Goals
- Clean and modular hardware design
- Fully open-source ecosystem
- Easy integration with existing 3D printer firmware (Klipper / Marlin)

## Notes
This project is experimental. Expect breaking changes in both hardware and design decisions.
